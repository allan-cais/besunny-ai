# Vite + React + TypeScript + Shadcn

This is a template for a web app using Vite, React, TypeScript, and Shadcn.

## Getting Started

1.  **Install dependencies**
    ```bash
    npm install
    ```
2.  **Run the dev server**
    ```bash
    npm run dev
    ```
3.  **Open the app**
    Open your browser to `http://localhost:5173`
